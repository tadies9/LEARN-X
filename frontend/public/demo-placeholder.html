LEARN-X Demo Video

Demo Flow (2 minutes max)

1. Upload PDF → 2. Ask Questions → 3. Get Insights

To create the actual demo video:

  1. Record a 2-minute screen capture showing:
    • PDF upload process
    • Chat interface in action
    • Educator dashboard insights
  2. Export as learn-x-demo.mp4
  3. Place in /frontend/public/
  4. Compress to ≤ 25 MB using:
    ffmpeg -i input.mov -c:v libx264 -preset slow -crf 23 -s 1920x1080 -r 30 learn-x-demo.mp4
