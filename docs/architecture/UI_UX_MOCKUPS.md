# LEARN-X UI/UX Mockups

## Design System Overview

### Color Palette
```
Primary Colors:
- Primary: #6366F1 (Indigo-500) - Main CTAs, active states
- Primary Dark: #4F46E5 (Indigo-600) - Hover states
- Primary Light: #818CF8 (Indigo-400) - Backgrounds

Neutral Colors:
- Gray-900: #111827 - Primary text
- Gray-700: #374151 - Secondary text
- Gray-500: #6B7280 - Muted text
- Gray-300: #D1D5DB - Borders
- Gray-100: #F3F4F6 - Backgrounds
- White: #FFFFFF - Cards, main background

Semantic Colors:
- Success: #10B981 (Emerald-500)
- Warning: #F59E0B (Amber-500)
- Error: #EF4444 (Red-500)
- Info: #3B82F6 (Blue-500)
```

### Typography
```
Font Family: Inter (System fallback: -apple-system, sans-serif)

Headings:
- H1: 36px/44px, Font-weight: 700
- H2: 30px/36px, Font-weight: 600
- H3: 24px/32px, Font-weight: 600
- H4: 20px/28px, Font-weight: 500

Body:
- Large: 18px/28px, Font-weight: 400
- Base: 16px/24px, Font-weight: 400
- Small: 14px/20px, Font-weight: 400
- Tiny: 12px/16px, Font-weight: 400
```

### Accessibility Features
- Focus rings: 2px offset, primary color
- Minimum touch target: 44x44px
- Color contrast: WCAG AA compliant
- Keyboard navigation indicators
- Screen reader announcements

## Design Principles Applied

1. **Familiar Patterns**: Layout similar to Canvas/Blackboard
2. **Progressive Disclosure**: Complex features revealed gradually
3. **Visual Hierarchy**: Clear primary, secondary, tertiary actions
4. **Consistent Spacing**: 8px grid system throughout
5. **Accessible Colors**: All combinations meet WCAG AA standards
6. **Mobile-First**: Every view optimized for mobile
7. **Error Prevention**: Inline validation, clear requirements
8. **Feedback**: Loading states, success messages, progress indicators

## Component Library

- Buttons: Primary, Secondary, Ghost, Disabled states
- Cards: Course cards, Module cards, Analytics cards
- Forms: Input fields, Textareas, Radio groups, Checkboxes
- Modals: Confirmation, Forms, Information
- Navigation: Top nav, Breadcrumbs, Tab navigation
- Feedback: Toast notifications, Progress bars, Loading spinners
- Data Display: Charts, Tables, Statistics
